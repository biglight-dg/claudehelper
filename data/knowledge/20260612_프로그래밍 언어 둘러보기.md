# 프로그래밍 언어 둘러보기 — Python · Java · JavaScript

## 🌍 프로그래밍 언어가 왜 여러 개일까

사람 말에 한국어·영어·일본어가 있듯, 컴퓨터에게 일을 시키는 언어도 여럿입니다. **목적에 따라 잘 맞는 언어가 다르기** 때문이에요.

**비유:** 운동도 목적별로 도구가 다릅니다. 달리기엔 운동화, 수영엔 수영복. 언어도 "이 일엔 이 언어"가 있습니다.

---

## 🐍 Python(파이썬) — 쉽고 만능

- **특징:** 사람 말과 가장 비슷해서 **배우기 쉬움**. 문장이 짧고 깔끔.
- **잘하는 분야:** AI·데이터 분석·자동화·업무 스크립트
- **이 교육 시스템도** 파이썬으로 만들어졌습니다.

**비유:** 누구나 쉽게 타는 자전거. 입문자에게 1순위로 추천되는 언어.

---

## ☕ Java(자바) — 튼튼한 대규모

- **특징:** 규칙이 엄격하고 **안정적**. 큰 회사의 대형 시스템에 많이 쓰임.
- **잘하는 분야:** 은행·기업용 시스템, 안드로이드 앱
- **주의:** 'Java'와 'JavaScript'는 **이름만 비슷할 뿐 완전히 다른 언어**입니다. (인도 vs 인도네시아처럼 헷갈리지만 별개)

**비유:** 튼튼하지만 다루는 데 면허가 필요한 대형 트럭.

---

## ⚡ JavaScript(자바스크립트) & Node.js

- **JavaScript:** 원래 **웹페이지에 동작**을 넣는 언어(앞서 배운 웹 3요소의 그 JS).
- **Node.js:** 그 JavaScript를 **웹페이지 밖, 서버에서도** 쓸 수 있게 해주는 도구. 덕분에 JS 하나로 화면(앞단)과 서버(뒷단)를 다 만들 수 있게 됐습니다.
- **잘하는 분야:** 웹사이트, 웹 서비스 전반

**비유:** 원래 집 안(브라우저)에서만 쓰던 도구를, Node.js가 집 밖(서버)에서도 쓰게 해준 셈.

---

## 📊 한눈에 비교

| 언어 | 난이도 | 대표 분야 | 비유 |
|---|---|---|---|
| Python | 쉬움 | AI·데이터·자동화 | 자전거 |
| Java | 어려움 | 기업·은행·안드로이드 | 대형 트럭 |
| JavaScript | 보통 | 웹 화면·동작 | 집 안 만능 도구 |
| Node.js | 보통 | 웹 서버 | 그 도구를 집 밖에서도 |

---

## 🧭 실무자가 기억할 것

- 언어 이름을 다 알 필요는 없습니다. **"이 일엔 보통 이 언어를 쓴다"** 정도면 충분.
- AI에게 자동화를 부탁하면 대개 **Python**으로 짜줍니다 — 그래서 이름 정도는 익숙해두면 좋아요.
- **Java ≠ JavaScript** 만큼은 꼭 기억하세요. 가장 흔한 오해입니다.

---

## 💡 핵심 교훈

- 언어가 여러 개인 이유는 **목적이 다르기** 때문 — 만능 1등은 없습니다.
- **Python = 쉽고 AI·자동화용**, **Java = 튼튼한 대규모용**, **JS/Node = 웹용**.
- **Java와 JavaScript는 완전히 다른 언어** — 이것만 헷갈리지 마세요.
