# 클로드 코드 설치와 셋업 — 검은 화면 두려움 없이 시작하기

## 🚀 클로드 코드가 뭐였더라

**클로드 코드(Claude Code)**는 채팅을 넘어 **내 컴퓨터에서 직접 파일을 만들고 고치고 실행하는** AI입니다. 단순 코딩 도구가 아니라 "AI와 함께 일하는 작업 환경(AI OS)"에 가깝죠. (개념은 'Claude Code·Cowork 입문' 참고)

이번엔 **실제로 깔고 켜는 법**입니다. 비개발자도 따라 할 수 있게, 막히는 지점까지 짚습니다.

---

## 💳 먼저, 구독 플랜 — 클로드 코드는 유료 전용

| 플랜 | 가격(월) | 누구에게 |
|---|---|---|
| Free | $0 | 클로드 코드 **사용 불가** |
| Pro | $20 | 입문자, 첫 실습 |
| Max (권장) | $100+ | 본격 사용·고사용량 |

**핵심:** 클로드 코드는 무료로는 못 씁니다. 가볍게 맛보려면 Pro($20), 업무에 제대로 쓰려면 Max를 권장합니다. (요금은 자주 바뀌니 가입 전 확인)

---

## 🖥️ 두 가지 사용 방법 — 터미널 vs IDE

| 방법 | 무엇 | 추천 대상 |
|---|---|---|
| **터미널(CLI)** | 검은 화면에 `claude` 입력 | 가볍고 빠름, 익숙한 사람 |
| **IDE 플러그인** | VS Code 등 안에서 사용 | 터미널이 낯선 **초보자에게 추천** |

**비유:** 터미널은 '맨손 조리', IDE는 '주방 도구가 갖춰진 환경'. 처음엔 IDE가 편하지만, 둘 다 결국 같은 클로드 코드입니다.

---

## ⚙️ 설치 전에 — Node.js와 Git, 두 친구

설치 방식에 따라 이 둘이 필요할 수 있어 미리 알아둡니다.

| 도구 | 한 줄 비유 | 왜 필요? |
|---|---|---|
| **Node.js** | 자바스크립트를 위한 '이동식 무대' | 클로드 코드가 그 위에서 돌아감(클래식 설치 시) |
| **Git** | 게임의 '세이브 파일' | AI가 망친 작업을 안전하게 되돌리는 안전망 |

**중요:** 클로드 코드는 **진짜 파일을 직접 고치는** 도구라, Git이 없으면 '세이브 없는 게임'과 같습니다. Git은 사실상 필수예요.

---

## 📦 설치 — 네이티브 권장 (한 줄로 끝)

운영체제별로 공식 설치 명령이 다릅니다. **방법 A(네이티브)**는 Node.js 없이 한 줄로 끝나 초보자에게 권장됩니다.

| OS | 네이티브 설치(권장) |
|---|---|
| 맥/리눅스 | 터미널에 공식 설치 명령 한 줄 실행 |
| 윈도우 | **PowerShell**에서 공식 설치 명령 실행 (CMD 아님) |

설치 후 공통 3단계: ① 프로젝트 폴더에서 `claude` 입력 → ② 브라우저로 로그인(한 번만) → ③ `claude doctor`로 상태 점검.

> ⚠️ **검토 요청:** 정확한 설치 명령·URL은 **공식 문서(code.claude.com/docs)에 항상 최신**으로 게시됩니다. 강의/수업 직전 반드시 공식 문서를 확인하세요.

---

## 🔧 막혔다면 — `claude doctor` 먼저

설치 후 가장 자주 막히는 지점과 처방입니다.

| 증상 | 처방 |
|---|---|
| `command not found` | **새 터미널 창** 열기 (경로 갱신 안 됨) |
| 윈도우 네이티브인데 안 잡힘 | 설치 경로가 PATH에 없음 — 안내대로 추가 |
| npm 권한 에러(EACCES) | **sudo 붙이지 말 것** (권한 꼬임 원인) |

당황하지 말고 `claude doctor`로 진단부터 — 대부분 '새 창 열기'로 해결됩니다.

---

## 🧩 더 쉬운 길 — 공식 확장(IDE)

VS Code 등에서는 **Anthropic 공식 확장**을 설치하면 더 편합니다. 단, 확장은 내 파일에 접근하므로 **반드시 공식 인증 마크**를 확인하세요.

- ✅ Extensions에서 'Claude Code' 검색 → **Anthropic 공식 + 인증 마크** 확인 후 설치
- ❌ 검증 마크 없는 비공식 확장 금지 (악성코드·파일 유출 위험)

---

## 💡 핵심 교훈

- 클로드 코드는 **유료 전용** — 입문 Pro($20), 본격 Max($100+). 사용법은 **IDE가 초보자에게 편함**.
- 처음이라면 **네이티브 설치 + Git 안전망**이 정답. 설치 명령은 **공식 문서가 항상 최신**.
- 막히면 **`claude doctor` → 새 터미널 창**. 확장은 **공식 인증 마크**를 꼭 확인하세요.
