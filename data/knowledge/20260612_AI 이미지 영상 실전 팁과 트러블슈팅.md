# AI 이미지·영상 제작 실전 팁 & 트러블슈팅

## 🧰 도구 비교 — 어떤 작업에 뭘 쓸까

이미지·영상 제작 도구는 많지만, **작업 종류별로 강한 도구**가 다릅니다.

| 작업 | 추천 도구 | 특징 |
|---|---|---|
| 이미지 생성 | Google 이미지(Nano Banana)·Midjourney | 고품질, 빠름 |
| 캐릭터 일관성 | 믹스보드 등 | 같은 인물 반복 생성 |
| 이미지→영상 | Kling AI·Google Flow | 정지컷에 움직임 부여 |
| 영상 편집 | CapCut | 자막·BGM·트랜지션 |

> ⚠️ **검토 요청:** 도구 이름·기능·요금은 매우 빠르게 바뀝니다. 수업 직전 최신 버전을 한 번 확인해 주세요.

---

## 🔧 결과가 이상할 때 — 증상별 처방

| 증상 | 원인 | 처방 |
|---|---|---|
| 손가락·얼굴이 이상함 | AI의 고질적 약점 | 부분 수정 기능으로 재생성, 손은 가리는 구도 |
| 매번 다른 캐릭터 | 일관성 정보 부족 | 캐릭터 속성값(JSON) 고정해 재사용 |
| 원하는 분위기 안 나옴 | 프롬프트 막연 | 5단계 공식 + 컬러/무드 구체화 |
| 영상이 뚝뚝 끊김 | 움직임 과함 | 카메라 무브 단순하게, 속도 낮춤 |
| 톤이 제각각 | 기준 색·무드 없음 | 전체 톤앤매너 먼저 정하고 시작 |

---

## 💡 품질을 끌어올리는 5가지 습관

1. **레퍼런스 먼저** — 원하는 느낌의 예시 이미지를 정해두고 시작
2. **한 번에 완성 금지** — 생성 → 부분 수정 → 재생성 반복
3. **프롬프트 저장** — 잘 나온 프롬프트는 메모해 재사용
4. **톤앤매너 고정** — 색·분위기 기준을 먼저 정하면 전체가 통일됨
5. **버전 남기기** — 마음에 든 결과물은 따로 보관 (덮어쓰기 방지)

---

## 🎬 이미지에서 영상까지 — 전체 흐름 한눈에

```
프롬프트(5단계) → 이미지 생성 → 부분 수정
   → 캐릭터/세계관 고정 → 스토리보드
   → Image-to-Video(Kling 5칸 공식) → 클립들
   → CapCut 편집(자막·BGM) → 최종 영상
```

각 단계의 결과물을 **다음 단계 입력으로** 넘기는 것이 핵심입니다.

---

## 💡 핵심 교훈

- 도구는 **작업 종류별로** 골라 쓰기 — 만능 하나는 없습니다.
- 결과가 이상하면 당황 말고 **증상별 처방표**대로 대응하세요.
- 품질의 비결은 재능이 아니라 **반복 수정 + 톤앤매너 고정 + 프롬프트 저장** 습관입니다.
