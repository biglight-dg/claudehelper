# 클로드 코드 설정과 .claude 디렉터리 — AI의 두뇌실 꾸미기

## 🧠 클로드 코드는 '설정 파일'로 움직인다

클로드 코드는 프로젝트 폴더 안의 **설정 파일들**을 읽고 그대로 따릅니다. 이 파일들이 모여 있는 곳이 **`.claude` 디렉터리** — 말하자면 클로드의 '두뇌실'이에요. ('AI 지침과 .md 파일 활용법'에서 배운 CLAUDE.md가 여기 살고 있습니다.)

---

## 📄 설정 파일은 두 가지 형식 — JSON과 Markdown

| 형식 | 누가 읽기 좋나 | 쓰임 | 생김새 |
|---|---|---|---|
| **.md (마크다운)** | 사람이 읽기 좋음 | 클로드 **지침서** | `#` 헤더, `-` 리스트 |
| **.json (제이슨)** | 기계가 읽기 좋음 | 클로드 **설정값** | `{ }` 중괄호, `,` 콤마 |

**핵심:** 사람에게 말하듯 적는 지침은 **.md**(CLAUDE.md, SKILL.md), 기계용 설정값은 **.json**(settings.json, .mcp.json). 둘은 역할이 다릅니다.

---

## 📜 CLAUDE.md — AI에게 주는 '운영 계약서'

`CLAUDE.md`는 "이 프로젝트에선 이렇게 일해줘"를 적어두는 지침 파일입니다. 클로드가 작업할 때마다 자동으로 읽어요.

```
# 프로젝트 컨텍스트
이 프로젝트는 카페 운영용입니다.

## 주의
- 결제 코드는 수정 금지
- 거래처는 존댓말로 응대
```

**비유:** 신입에게 주는 '업무 매뉴얼 + 사규'. 한 번 적어두면 매번 설명할 필요가 없습니다. (`/init` 명령으로 초안을 자동 생성할 수도 있습니다.)

---

## 🗂️ .claude 디렉터리 구조 — 무엇이 들어있나

```
.claude/
├── settings.json         권한·훅 설정 (팀 공유)
├── settings.local.json   내 PC 전용 설정
├── CLAUDE.md             프로젝트 지침 (팀 공유)
├── CLAUDE.local.md       내 전용 지침
├── agents/               서브에이전트
├── skills/               반복 작업(스킬)
├── rules/                분리한 규칙들
└── .mcp.json             MCP(외부 도구) 설정
```

| 파일 종류 | 공유 범위 |
|---|---|
| `settings.json` · `CLAUDE.md` | **팀 공유** (Git에 올라감) |
| `.local` 붙은 파일 | **내 컴퓨터 전용** (업로드 차단) |

**핵심:** 파일명에 **`.local`이 붙으면 나만 쓰는 개인 설정** — 팀에 안 올라갑니다. 비밀·개인 취향은 `.local`에.

---

## 🔌 확장 기능 4가지 — 클로드를 키우는 도구들

`.claude` 안에서 클로드의 능력을 넓히는 4가지입니다. (각각은 뒤 과정에서 깊게 다룹니다.)

| 기능 | 한 줄 정의 | 예시 |
|---|---|---|
| **MCP** | 외부 도구 연결 규격 | Gmail·Notion·Slack을 클로드에 연결 |
| **Skill** | 반복 업무 절차서 묶음 | "회의록 정리"를 매번 같은 방식으로 |
| **Subagent** | 별도 보조 에이전트 | Research·QA 등 역할 분담·병렬 처리 |
| **Hook** | 자동 실행 스크립트 | "작업 끝나면 알림" 자동 검증 |

**비유:** 클로드라는 일꾼에게 — 연장통(MCP), 업무 매뉴얼(Skill), 보조 직원(Subagent), 자동 반응 스위치(Hook)를 달아주는 것.

---

## 💡 핵심 교훈

- 클로드 코드는 **`.claude` 디렉터리의 설정 파일**을 읽고 움직입니다 — 클로드의 '두뇌실'.
- **지침은 .md(CLAUDE.md=운영 계약서), 설정값은 .json**(settings.json). `.local`이 붙으면 나만 쓰는 개인용.
- 능력 확장 4종 — **MCP(연결)·Skill(반복 절차)·Subagent(분담)·Hook(자동 실행)**. 이름과 역할만 잡아두면 충분합니다.
