# Claude Code와 Claude Cowork — AI가 내 컴퓨터에서 직접 일한다

## 🦾 채팅형 AI의 한계부터

ChatGPT·Claude 같은 채팅형 AI는 똑똑하지만 **"말로 알려줄 뿐"** 입니다. "보고서 폴더 정리해줘"라고 하면 방법을 설명해주지만, **실제로 내 컴퓨터의 파일을 정리해주지는 못해요.**

**비유:** 전화로 길을 알려주는 친구. 친절하지만, 내 손을 잡고 직접 데려다주지는 않습니다.

여기서 한 단계 진화한 것이 **에이전트형 AI** — 직접 손발을 움직여 일을 처리합니다. 대표가 **Claude Code**와 **Claude Cowork**입니다.

---

## ⌨️ Claude Code — 터미널에서 일하는 AI

**Claude Code**는 터미널(검은 화면) 안에서 동작하는 AI입니다. 내 컴퓨터의 파일을 **직접 읽고, 만들고, 고치고, 정리**할 수 있어요. (지금 이 교육 시스템도 Claude Code가 운영하고 있습니다.)

할 수 있는 일:
- 폴더·파일 정리, 이름 일괄 변경
- 문서 자동 생성 (이 과정의 교재·슬라이드처럼)
- 자료 검색·요약·정리
- 반복 작업 자동화

**비유:** 말귀를 알아듣고 **직접 책상까지 정리해주는** 비서. 한국어로 "inbox 폴더 정리해줘"라고 하면 알아서 처리합니다.

---

## 🤝 Claude Cowork — 옆에서 함께 일하는 AI

**Claude Cowork**는 "Cowork = Co(함께) + Work(일하다)"라는 이름처럼, **옆자리 동료처럼 협업**하는 형태의 Claude입니다. 복잡한 작업을 단계별로 같이 진행하고, 여러 자료와 도구를 오가며 일을 거들어 줍니다.

> ⚠️ **검토 요청:** Claude Cowork의 정확한 화면·세부 기능·요금은 빠르게 업데이트되는 영역입니다. 수업 직전 최신 공식 안내를 한 번 확인해 주세요.

**Claude Code vs Cowork 한눈에:**

| 구분 | Claude Code | Claude Cowork |
|---|---|---|
| 무대 | 터미널(검은 화면) | 함께 일하는 협업 공간 |
| 느낌 | 명령하면 처리하는 실행 비서 | 옆에서 같이 고민하는 동료 |
| 잘 맞는 일 | 파일·자동화·개발 작업 | 자료 검토, 단계적 협업 작업 |

**비유:** Claude Code가 "시키면 척척 해내는 실무 직원"이라면, Cowork는 "어깨너머로 같이 보면서 같이 만들어가는 짝꿍"입니다.

---

## 🧭 실무자가 기억할 핵심

- 채팅형(설명만) → 에이전트형(직접 실행)으로 **AI의 손발이 생겼다**는 것이 큰 변화입니다.
- 둘 다 **한국어로 평범하게 부탁**하면 됩니다. 명령어를 외울 필요 없어요.
- 다만 AI가 내 파일을 직접 다루므로, **무엇을 하려는지 먼저 확인하고 실행**하는 습관이 중요합니다. (안전 과정에서 더 다룹니다)

---

## 💡 핵심 교훈

- **Claude Code** = 터미널에서 내 컴퓨터 파일을 직접 다루는 실행형 AI.
- **Claude Cowork** = 동료처럼 함께 작업을 진행하는 협업형 Claude.
- 공통점: "말로 설명"을 넘어 **"직접 일을 처리"**한다는 것 — 실무 자동화의 출발점.
