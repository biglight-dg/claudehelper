# Claude Code·Cowork 실전 사용 — 한국어로 시켜서 진짜 일 끝내기

## 🎯 개념은 배웠으니, 실제로 시켜봅니다

'AI 업무 기초'에서 Claude Code·Cowork가 **내 컴퓨터에서 직접 일하는 에이전트형 AI**라는 건 배웠습니다. 이번엔 **실제로 어떤 말로 시키고, 무엇이 나오는지** 구체적으로 봅니다. 명령어는 외울 필요 없어요 — **평범한 한국어**면 됩니다.

**비유:** 유능한 비서에게 메모를 건네듯 말하면 됩니다. "이거 정리해서 보고서로 만들어줘"처럼요.

---

## ⌨️ Claude Code — 실제 한국어 명령 예시

진짜로 이렇게 입력하면 됩니다. 입력(내가 시키는 말) → 결과(AI가 하는 일)로 봅시다.

| 내가 시키는 말 (한국어 그대로) | Claude Code가 하는 일 |
|---|---|
| "inbox 폴더의 메모들을 회의록 .md로 정리해줘" | 메모를 읽고 회의록 양식의 .md 파일 생성 |
| "이 폴더 파일 이름을 날짜_제목 형식으로 바꿔줘" | 파일명을 일괄 변경 |
| "지난달 매출 엑셀에서 핵심만 표로 요약해줘" | 파일을 읽고 요약 표 작성 |
| "이 보고서 오타랑 어색한 문장 고쳐줘" | 문서를 직접 수정 |

**핵심:** 명령어·코드가 아니라 **"무엇을, 어떻게"를 한국어로** 부탁하는 것. 이 교육 시스템의 교재·슬라이드도 Claude Code가 이렇게 만듭니다.

---

## 🤝 Claude Cowork — 함께 단계로 진행하기

Code가 "시키면 척척 처리"라면, Cowork는 **옆에서 같이 고민하며 단계로** 진행합니다. 한 번에 안 끝나는 복합 작업에 맞아요.

예시 흐름:
1. "신제품 출시 계획을 같이 짜보자" (내가 운을 뗌)
2. Cowork가 초안 구조를 제안 → 내가 의견
3. 자료를 같이 보며 단계별로 다듬어 완성

**비유:** Code는 "심부름을 끝내오는 직원", Cowork는 "회의실에서 같이 화이트보드를 채우는 동료".

## 📊 Code vs Cowork — 언제 무엇을

| 상황 | 추천 | 이유 |
|---|---|---|
| 파일 정리·이름 변경·자동화 | Claude Code | 정해진 작업을 빠르게 처리 |
| 문서 일괄 생성·수정 | Claude Code | 내 컴퓨터 파일을 직접 다룸 |
| 기획·전략을 함께 다듬기 | Claude Cowork | 단계적 협업·토론에 강함 |
| 자료 검토하며 결론 내기 | Claude Cowork | 옆에서 같이 진행 |

## ⚠️ 실전 안전 습관 — '실행 전 확인'

AI가 **내 파일을 직접 바꾸기 때문에**, 편리한 만큼 조심도 필요합니다.

- 중요한 폴더는 작업 전 **백업/복사본**을 두기
- "삭제·덮어쓰기"가 들어가는 작업은 **무엇을 할지 먼저 확인**하고 실행
- 처음엔 **작은 폴더·사본**으로 연습한 뒤 실제 업무에 적용

> ⚠️ **검토 요청:** Claude Code·Cowork의 화면·설치·요금·세부 기능은 빠르게 업데이트됩니다. 수업 직전 최신 공식 안내를 확인해 주세요.

## 💡 핵심 교훈

- 둘 다 **평범한 한국어로 부탁**하면 됩니다 — 명령어 암기 불필요.
- **Code = 시키면 처리하는 실행 비서 / Cowork = 함께 다듬는 협업 동료.** 작업 성격으로 고르세요.
- 편리한 만큼 **'실행 전 확인'**이 필수 — 작은 사본으로 먼저 연습하세요.
