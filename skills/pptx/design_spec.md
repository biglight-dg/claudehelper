# 디자인 명세 — 흑백 미니멀 (교육팀 기준)

교육자 에이전트가 PPT를 만들 때 반드시 따르는 디자인 규칙.

---

## 색상 팔레트 (흑백 전용)

| 역할 | 색상 | HEX |
|------|------|-----|
| 배경 | 흰색 | `FFFFFF` |
| 제목 텍스트 | 검정 | `000000` |
| 본문 텍스트 | 진한 회색 | `222222` |
| 보조 텍스트 (라벨, 캡션) | 중간 회색 | `666666` |
| 구분선 / 테두리 | 연한 회색 | `CCCCCC` |
| 강조 (꼭 필요할 때만) | 검정 박스 흰 텍스트 | `000000` / `FFFFFF` |

**금지사항**: 파란색·빨간색·초록색 등 유채색 사용 금지. 그라디언트 금지. 그림자 금지.

---

## 폰트

| 용도 | 폰트 | 크기 | 굵기 |
|------|------|------|------|
| 슬라이드 제목 | Pretendard | 36pt | Bold |
| 본문 불릿 | Pretendard | 20pt | Regular |
| 섹션 라벨 (상단 소제목) | Pretendard | 14pt | Regular |
| 캡션 / 출처 | Pretendard | 12pt | Regular |

폴백 순서: `Pretendard` → `Apple SD Gothic Neo` → `Malgun Gothic` → `Arial`

---

## 레이아웃 규칙

- 여백: 상하좌우 최소 0.5인치 (약 1.3cm)
- 슬라이드 크기: 16:9 (10인치 × 5.625인치)
- 구분선: 가로 1pt 회색 선 (`CCCCCC`)
- 정렬: 텍스트는 왼쪽 정렬 (중앙 정렬은 타이틀 슬라이드만)

---

## 슬라이드 타입별 레이아웃

### 타이틀 슬라이드
```
┌─────────────────────────────────┐
│                                 │
│   (중앙 정렬)                    │
│   제목 (36pt Bold)              │
│   ────────────────              │
│   부제목 (20pt Regular #666)    │
│                                 │
│                     AI 교육팀   │  ← 우하단, 12pt #666
└─────────────────────────────────┘
```

### 콘텐츠 슬라이드
```
┌─────────────────────────────────┐
│  섹션 라벨 (14pt #666)           │
│  ─────────────────────────────  │  ← CCCCCC 구분선
│  슬라이드 제목 (36pt Bold)       │
│                                 │
│  • 불릿 항목 1 (20pt)           │
│  • 불릿 항목 2                  │
│  • 불릿 항목 3                  │
│                                 │
└─────────────────────────────────┘
```

### 비교 슬라이드
```
┌─────────────────────────────────┐
│  비교 제목 (36pt Bold)           │
│  ─────────────────────────────  │
│  왼쪽 제목  │  오른쪽 제목       │  ← Bold 20pt
│  • 항목    │  • 항목            │
│  • 항목    │  • 항목            │
└─────────────────────────────────┘
```

### 요약 슬라이드
```
┌─────────────────────────────────┐
│  핵심 교훈 (36pt Bold)           │
│  ─────────────────────────────  │
│  1. 교훈 내용                   │
│  2. 교훈 내용                   │
│  3. 교훈 내용                   │
│                                 │
│  출처: ...  (12pt #666)         │
└─────────────────────────────────┘
```

## python-pptx 색상 설정법

```python
from pptx.util import Pt
from pptx.dml.color import RGBColor

# 색상 예시 (# 없이 사용)
BLACK = RGBColor(0x00, 0x00, 0x00)
GRAY_DARK = RGBColor(0x22, 0x22, 0x22)
GRAY_MID = RGBColor(0x66, 0x66, 0x66)
GRAY_LIGHT = RGBColor(0xCC, 0xCC, 0xCC)
WHITE = RGBColor(0xFF, 0xFF, 0xFF)

# 폰트 설정
run.font.name = "Pretendard"
run.font.size = Pt(36)
run.font.bold = True
run.font.color.rgb = BLACK
```
