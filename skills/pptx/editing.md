# 기존 PPTX 편집 워크플로우

기존 .pptx 파일을 수정할 때 사용하는 XML 기반 워크플로우.

## 7단계 프로세스

1. **템플릿 분석** — thumbnail.py로 슬라이드 구조 파악
2. **슬라이드 매핑** — 어떤 슬라이드를 남기고 추가할지 계획
3. **압축 해제** — `Expand-Archive file.pptx unpacked/ -Force`
4. **구조 변경** — 슬라이드 삭제/복제/순서 변경 (add_slide.py 사용)
5. **내용 편집** — 개별 슬라이드 XML 직접 수정 (Edit 도구)
6. **정리** — `python skills/pptx/scripts/clean.py unpacked/`
7. **재압축** — `Compress-Archive unpacked/* output.pptx`

## 핵심 디자인 원칙

**단조로운 슬라이드 피하기** — 타이틀+불릿 구조만 반복하지 말 것.
다양한 레이아웃 사용: 2단 구성, 전면 이미지, 인용 슬라이드, 구분자 슬라이드.

## XML 편집 규칙

- 내용 변경은 sed/awk 대신 **Edit 도구** 사용
- 헤더와 인라인 레이블은 **굵게**
- 여러 항목은 별도 단락으로 분리 (한 줄에 이어 붙이지 말 것)
- 불필요한 요소는 텍스트만 지우지 말고 **완전히 삭제**
- 스마트 따옴표: `&#x201C;` (좌) `&#x201D;` (우)

## 주의사항

구조 변경(삭제/추가) 완료 후 내용 편집 시작.
서브에이전트로 병렬 편집 가능 (슬라이드별로 분리).
